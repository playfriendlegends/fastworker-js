# create-fastworker

The official CLI scaffolding tool for `fastworker-js` projects.

- **Website & Documentation**: [https://playfriendlegends.github.io/fastworker-js/](https://playfriendlegends.github.io/fastworker-js/)
- **GitHub Repository**: [https://github.com/playfriendlegends/fastworker-js](https://github.com/playfriendlegends/fastworker-js)

## Quick Start

Initialize a new `fastworker-js` project in one command:

```bash
npx create-fastworker@latest my-app
```

The wizard will guide you through choosing:
- **Language**: TypeScript or JavaScript
- **Topology**: Monolith or Microservices
- **Deployment Platform**: Cloudflare Workers or Node.js

Once initialized, navigate to your project and start the development server:

```bash
cd my-app
npm install
npm run dev
```
